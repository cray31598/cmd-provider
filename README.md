# cmd-provider

Node.js backend API boilerplate (Express).

## Setup

```bash
npm install
cp .env.example .env
```

## Run

```bash
npm run dev    # development with auto-reload
npm start      # production
```

## Endpoints

- `GET /health` – health check
- `GET /api` – API root

Default port: **3000** (override with `PORT` in `.env`).
